API Documentation

@POST
This service doesn't has any parameter. It has a submit button called "Buy your products". This will post payment.

@GET
This service will get all the payments but this doesnt have any paramteres.

@GET
"/findID/{id}"
This service has a parameter. This will get the payment of the particular id.

@Delete
"/deletePayment/{id}" This service has a parameter. This will delete payment of the particular id.
